Karidanse (retour) 6c+ 7a Le Calvaire

-
traverse frtl, crimps, sitstart

Sitstart without crash-pad under Le Cube (orange n°14), traverse to the left without the top holds and exit with the left arete.
Update (March 2025) : the starting hold left hand has partially broken off but the problem is still possible with the same grade.
