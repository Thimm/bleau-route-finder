La Jouissance du Masséter 7a+ Cuisinière

-
arete, crimps

Underneath La Rafale (white n°7), along the path towards Duel, from the red circuit. Start with the obvious lip low and exit via the arete.
NB: the name of this boulder is due to an old error because "La Jouissance du Masséter" was given at the beginning to the problem now called "Rencard".
