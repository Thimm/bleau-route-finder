Liakyssan 7b+ Cuvier Bellevue

Marc LEHNER
traverse frtl, overhang, sitstart

Sitstart with the big hold, low traverse to the left and exit as Anak.
NB: the first ledge of this problem moves a lot and threatens to break, so you should no longer use it as a hold and start with the ledge above.
